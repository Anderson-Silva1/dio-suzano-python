# **Modo interativo**

O interpretador Pytho permite ao desenvolvedor escrever código e ver o resultado na mesma hora

## Temos duas formas de usaro mode interativo:

1. Chamando o próprio interpretador Python

2. Execultando o scrit com a flag -i (python -i app.py)

## Função dir e help

### dir

O dir retorna para nós a listade métodos que podemos usar com determinado tipo.

Sem argumentos, ela retorna a lista de nomes no escopo local atual. Com um argumento retorna uma lista de atributos válidos parao objeto. Exemplo:

dir()

```python
['__annotations__', '__builtins__', '__doc__', '__loader__', '__name__', '__package__', '__spec__']
```

### help

Invocao sistema de ajuda integrado do Python. É possível fazer buscas em modo interativo, ou informar por parâmetro qual o nome do módulo, função, classe, método ou variável.

Exemplo:

help()

help(100)

Para sair dele usamos a letra "q"
